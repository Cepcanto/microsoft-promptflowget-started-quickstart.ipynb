# How to automatically generate the REST client code
Rest client code in this folder are not manually written, but generated by openapi-generator-cli.

## Setup
+ install [nodejs](https://nodejs.org/en)
+ install swagger2openapi
  + run `npm install -g swagger2openapi`
+ install openapi-python-client
  + run `pip install openapi-generator-cli`

## Generate python rest client
1. Update swagger.json

    Update `../service/swagger.json` with the latest PFS APIs.
2. Convert Swagger 2.0 to OpenAPI 3.0

    Use a tool like swagger2openapi to convert Swagger 2.0 specification to OpenAPI 3.0.
    ```
    swagger2openapi ../_service/swagger.json -o openapi.yaml
    ```

3. Generate python rest client
    ```python
    openapi-python-client generate --path ./openapi.yaml --meta none --config ./client_config.yaml --custom-template-path=./template
    ```
